# Controle de Pregões

Sistema para organizar fichas de controle de pregões.

---

## Estrutura do projeto

### server.js
Ponto de entrada da aplicação Node.
Responsável por iniciar o servidor e carregar a aplicação.

### src/app.js
Arquivo de configuração da aplicação.
Aqui configuramos Express, rotas e middlewares.

### src/controllers
Contém a lógica da aplicação.
Responsável por processar dados e responder às requisições.

### src/routes
Define as rotas do sistema.
Cada rota indica qual controller deve ser executado.

### src/views
Contém as páginas da aplicação (interface do usuário).

### src/public
Arquivos públicos acessíveis pelo navegador.

Exemplos:
- CSS
- JavaScript do frontend
- imagens
